AirQuality APP

En este directorio se encuentra la alicación para smartphones Android que se encarga de escuchar 
los beacons enviados por el dispositivo de medición de calidad de aire con las lecturas que va
tomando y enviarlas al servidor para que se incluyan en la base de datos.

Está escrito en el lenguaje Java.

Para instalar la aplicación en un terminal bastará con abrir el proyecto (el directorio 
"AirQuality") en Android Studio e instalarlo haciendo "click" en el botón de "play" de 
color verde que aarecerá en la parte superior junto al nombre del terminal seleccionado.
